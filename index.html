Introduction

If you find some public data for your data analysis, you may find some data from twitter. But I found the process using twitter API is little bit difficult to us.
You should know not only the knowledge of the API, but also programming skills. But I know, many of you guys don't have experience of programming.
That's why this project began. I just want to make it easier and less stress to get public data from twitter.
As you know, twitter has a lot of good stuff for data analysis. Not a few media analyze the tweets of US presidential candidates!
You can catch the treanding topics, events, lifestyles, ecomonics, politics and etc from twitter.
Now, all you need to is your twitter account and developer key from the twitter.
